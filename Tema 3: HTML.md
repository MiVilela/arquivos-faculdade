## HTML

### Títulos HTML

Os títulos são definidos com as tags **<h1>** até a **<h6>**
H1 define o título mais importante e o h6 o menos importante, tendo diferença em tamanho de fonte.

### Paragráfos

Os parágrafos são definidos com a tag **<p>**
